# 💎 Gioncyn Vision - Premium PWA Converter

**Gioncyn Vision** adalah aplikasi web berbasis Progressive Web App (PWA) yang dirancang untuk memberikan pengalaman konversi format gambar yang cepat, elegan, dan profesional. Dengan antarmuka yang modern, aplikasi ini memungkinkan pengguna untuk mengubah gambar ke format PNG, JPG, atau WEBP secara instan.

---

## ✨ Fitur Utama
* **Elite UI/UX**: Desain antarmuka mewah dengan gradasi warna ungu-biru yang elegan.
* **Multi-Format Support**: Konversi gambar ke format PNG, JPG, dan WEBP dengan satu klik.
* **PWA Ready**: Dapat diinstal di layar beranda HP atau Desktop dengan ikon **VG Gold** yang eksklusif (Tampilan aplikasi layaknya APK asli).
* **Offline Mode**: Berkat teknologi Service Worker, aplikasi tetap dapat diakses meski tanpa koneksi internet.

## 🛠️ Teknologi yang Digunakan
* **HTML5 & CSS3**: Untuk struktur dan gaya visual yang responsif.
* **JavaScript (Vanilla)**: Menangani logika pemrosesan gambar dan Service Worker.
* **JSON**: Digunakan pada file Manifest untuk integrasi ikon aplikasi seluler.

## 📂 Struktur Proyek
* `index.html` - Antarmuka pengguna dan logika utama aplikasi.
* `manifest.json` - Konfigurasi ikon dan identitas aplikasi untuk instalasi PWA.
* `sw.js` - Service Worker untuk mendukung penggunaan offline dan cache.
* `logo-vg.png` - Aset ikon VG Gold resmi (Pastikan file gambar menggunakan nama ini).

---

## 🚀 Cara Instalasi di HP
1. Buka link **GitHub Pages** proyek ini di browser Chrome Android.
2. Tunggu hingga halaman termuat sepenuhnya.
3. Klik ikon **titik tiga (⋮)** di pojok kanan atas Chrome.
4. Pilih menu **"Instal aplikasi"** (ikon logo VG Gold akan muncul).
5. Aplikasi sekarang tersedia di menu utama HP Anda.
